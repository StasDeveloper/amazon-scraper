# amazon-scraper
Tech stack:
- Laravel
- Guzzle HTTP client to get page content
- User Agent to emulate various browsers
- Crawel to work with DOM

Instal notes:
Run project with 
``` bash
composer install
```
Usage:
<host_name>/api/parser?link=https://www.amazon.co.uk/Winning-Moves-29612-Trivial-Pursuit/dp/B075716WLM/
